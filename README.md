# 家庭簿アプリ (KakeiboGX)

Android向けの多機能家計簿アプリケーションです。日々の支出・収入の記録から、カレンダーごとの収支確認、カテゴリ別のグラフィカルな分析レポート、予算（クォータ）管理、プリセット入力、CSVエクスポート/インポートなど、家計管理に必要な一連の機能を備えています。

## 主な機能一覧

### 1. 入力画面（Home / Input）
- **支出 / 収入の記録**: タブで直感的に切り替え可能。
- **日付選択**: カレンダー専用ダイアログから過去や未来の日付を簡単に指定。
- **金額入力（ビルトイン電卓）**: ボトムシート型の電卓UIで入力可能。
- **カテゴリ選択**: アイコンと色が設定されたカテゴリリストから選択。
- **⚡ プリセット入力**: よく使うメモと金額のセットをプリセット登録し、ワンタップで呼び出し可能。使用頻度順に自動ソートされ、よく使うものが上に表示されます。
- **ドラッグ＆ドロップ並べ替え**: カテゴリを長押しでドラッグし、好みの順番に並べ替え可能。並び順はDB保存され永続化されます。

### 2. カレンダー画面（Calendar）
- **月間カレンダー表示**: 日ごとに収入（青字）と支出（赤字）をひと目で確認。月〜日の曜日ヘッダー付き。
- **日別データリスト**: その月の全記録を年月日付きでリスト表示。項目タップでデータの編集・削除が可能。
- **月間サマリ**: 月の総収入・総支出・収支計算を表示。

### 3. レポート画面（Report）
- **カテゴリ別割合（円グラフ）**: 支出（または収入）のカテゴリ別内訳を視覚化。スライスタップで詳細表示。
- **カテゴリリスト**: 合計金額・割合を表示。長押しドラッグで並べ替え可能。
- **予算（クォータ）表示**: カテゴリごとに残額・今日の目標額・今日の使用額・あと使える額を表示。今日の目標は「昨日までの実績ベース」で計算されるため、支出しても目標額がブレません。
- **カテゴリ別月間レポート**: タップすると棒グラフ付き詳細画面へ遷移。全期間の月別推移を表示し、今月が中央に来るよう自動スクロール。

### 4. その他・設定画面（Settings）
- **カテゴリの管理・編集**: カスタムカラーでカテゴリを自由に追加・編集。
- **固定収支の設定**: 毎月固定の支出（家賃、サブスク等）や収入を登録。
- **予算（クォータ）設定**: カテゴリごとに毎月の予算を設定。
- **CSVバックアップ**: 全データをCSVとしてエクスポート/インポート。他アプリ（sekkei.md互換フォーマット）からのインポートにも対応し、存在しないカテゴリは自動作成されます。

## 技術スタック
| 項目 | 技術 |
|---|---|
| 言語 | Kotlin |
| アーキテクチャ | MVVM + Repository パターン |
| UI | XML Layouts + ViewBinding + Material Components |
| ナビゲーション | Jetpack Navigation Component (BottomNavigationView) |
| DB | Room Database (v2, マイグレーション対応) + Coroutines / Flow |
| グラフ | MPAndroidChart (円グラフ・棒グラフ) |

## データベース構成
| テーブル | 概要 |
|---|---|
| `categories` | カテゴリ（名前・色・タイプ・表示順） |
| `daily_data` | 日次データ（日付・金額・メモ・カテゴリ・タイプ） |
| `fixed_cost_settings` | 固定費設定 |
| `quota_settings` | 予算設定 |
| `presets` | プリセット（メモ・金額・使用回数） |

## ビルド方法
1. Android Studio でプロジェクトを開く
2. Gradle Sync を実行
3. エミュレータまたは実機で `Run 'app'` を実行
   - **動作環境**: Android 8.0 (API 26) 以上
