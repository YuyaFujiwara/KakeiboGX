# 家庭簿アプリ (KakeiboGX)

Android向けの多機能家計簿アプリケーションです。日々の支出・収入の記録から、カレンダーごとの収支確認、カテゴリ別のグラフィカルな分析レポート、予算（クォータ）管理、CSVへのエクスポート/インポートなど、家計管理に必要な一連の機能を備えています。

## 主な機能一覧

### 1. 入力画面（Home / Input）
- **支出 / 収入の記録**: タブで直感的に切り替え可能。
- **日付選択**: カレンダー専用ダイアログから過去や未来の日付を簡単に指定。
- **金額入力（ビルトイン電卓）**: 画面下部から出現するボトムシート型の電卓UIを利用して入力可能。
- **カテゴリ選択とメモ**: アイコンと色が設定された分かりやすいカテゴリリストから選択可能。

### 2. カレンダー画面（Calendar）
- **月間カレンダー表示**: 日ごとに収入（青字）と支出（赤字）をひと目で確認。
- **日別データリスト**: その月の全記録をリスト表示。項目をタップするとデータの編集・削除が直接行えます。
- **月間サマリ**: 月の総収入・総支出・最終的な収支計算を表示。

### 3. レポート画面（Report）
- **カテゴリ別割合（円グラフ）**: その月の全支出（または収入）がどのカテゴリで使われたかを視覚的に分析。
- **詳細リスト表示**: カテゴリごとの合計金額と全体に占める割合（％）をリスト形式で詳細に表示。
- **支出・収入切り替え**: 簡単なタブ操作で支出分析と収入分析を切り替え。

### 4. その他・設定画面（Settings）
- **カテゴリの管理・編集**: プリセットカラーの中から好きな色を選び、あなただけのカスタムカテゴリを自由に追加可能。
- **固定収支の設定**: 毎月固定で発生する支出（家賃、サブスク費用など）や収入を登録可能。
- **予算（クォータ）設定**: 食費や交際費などカテゴリごとに毎月の予算を設定。
- **CSVバックアップ機能**: 記録した全データをCSVとしてエクスポート（書き出し）、またはインポート（読み込み）可能。他端末やPCでのデータ管理にも対応。

## 技術スタック (Architecture & Libraries)
- **言語**: Kotlin
- **アーキテクチャ**: MVVM (Model-View-ViewModel) + Repository パターン
- **UI フレームワーク**: XML Layouts + ViewBinding + Material Components
- **ナビゲーション**: Jetpack Navigation Component (BottomNavigationView)
- **ローカルデータベース**: Room Database + Kotlin Coroutines / Flow 
- **グラフ描画**: MPAndroidChart (円グラフ)

## インストール・ビルド方法
1. Android Studio で当プロジェクトを開きます。
2. グレードルの同期（Sync Project with Gradle Files）を行います。
3. エミュレータ、または実機を接続し「Run 'app'」を実行します。
    - **動作環境**: Android 8.0 (API Level 26) 以上を推奨
