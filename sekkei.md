# アンドロイドの家計簿アプリを作る。

# 基本
下に4つ項目がある
1.入力
2.カレンダー
3.レポート
4.その他


# 入力UI
1. 支出、収入の2タブがある。

2. その下に日付がある。左右に矢印ボタンがあって、翌日前日に移動でき、カレンダーアイコンをクリックするとカレンダーで指定の日付にできる。
    - HHHH/MM/DD/(曜日)形式。

3. その下にメモ。要するに何に使ったか。タップするとキーボードが開く

4. その下に支出。タップすると電卓っぽいUIが開く。

    - 1~4までは画面に固定されているUI

5. その下にカテゴリーがある。正方形の中にアイコンがあって、支出に対して1つ選んでホールドする。下にスクロールできるようにする。
    - カテゴリーは別途編集画面で追加できるようにする。

6. 下に支出を入力するボタンがあり、タップすると入力した内容を支出として登録する。

7. 収入は1~5と同様。
## 詳細UI
1~4までの行で上から40%ぐらい、5が60％ぐらい。


# カレンダーUI
1. １行目は今月が出る(  YYYY/MM)。左右に矢印ボタンがあって、前月翌月移動できる。
2. その下にカレンダーが出る。各セルに収入、支出がそれぞれ赤字青字で入る。収入がセルの上段、支出がセルの下段。
3. カレンダーの下に月間の収入、支出、合計(収支)が出る。
4. その下に日付け、その日の収支のリストがスクロール可能な行で出る
    - 例
    ```
    3月27日（金）          -142809円
    [アイコン] [カテゴリ] [メモ] [金額]
    [アイコン] [カテゴリ] [メモ] [金額]
    [アイコン] [カテゴリ] [メモ] [金額]
    3月26日（木）          +1149円
    [アイコン] [カテゴリ] [メモ] [金額]
    [アイコン] [カテゴリ] [メモ] [金額]
    [アイコン] [カテゴリ] [メモ] [金額]
    ```
    - リストの項目をタップすると編集画面に移動する。編集画面は入力画面と同様。
    - ただし、編集画面では支出を上書きするボタンの下に、コピー、削除がある
        - 削除は削除、コピーは内容をコピーして新しい入力画面を開く。
## 詳細UI
- カレンダーまでは半分ぐらい。
- 年月とカレンダーまで固定UI,月間収支以降はスクロールUI
# レポートUI
1. １行目はカレンダーUIと同様
2. 長方形を３つ割りにして左上が支出、赤文字で-X円、右上が収入青文字手+X円、下が収支、黒文字で+か-X円
3. その下に2つタブがあり、支出、収入がそれぞれカテゴリで色分けされて表示される。
    - 円グラフの各カテゴリ(色)をタップすると、カテゴリ名、金額、全体に占める割合がフキダシでオーバレイ表示される。
4. その下にカレンダーUIの4のカテゴリ版のリストが表示される。
    - 例
    ```
    [アイコン] [カテゴリ] [金額] [割合]
    [アイコン] [カテゴリ] [金額] [割合]
    [アイコン] [カテゴリ] [金額] [割合]
    ```
    - リストの項目をタップするとカテゴリ別の月間レポートに遷移。
    - カテゴリ別の月間レポートはカレンダーUIの4と同様のリスト表示。
    - ただし、この画面では固定UIの棒グラフがあり、棒グラフは左右にスクロールすることで、今まで入力した月間の記録が見える。
    - ただし、リストの項目をタップすると編集画面に移動する。編集画面は入力画面と同様。
    - ただし、編集画面では支出を上書きするボタンの下に、コピー、削除がある
        - 削除は削除、コピーは内容をコピーして新しい入力画面を開く。
## 詳細UI
- 1行目だけ固定UI、それ以降は全てスクロールUI
